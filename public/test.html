Member API

POST /member 회원 등록

Request Body

필드 타입 필수 설명
user_id string ✅ 유저 고유 ID
nickname string ✅ 닉네임 (중복 불가) 
{
  "user_id": "user-abc123",
  "nickname": "플레이어1"
}
201 Created 400 필수 필드 누락 400 중복 user_id / 닉네임
GET /member/{user_id} 회원 조회 
{
  "user_id": "user-abc123",
  "nickname": "플레이어1",
  "created_at": "2026-03-25T00:00:00.000Z"
}
200 OK 400 user_id 미입력 404 존재하지 않는 유저

Rank API

GET /rank Top 10 랭킹 
[
  {
    "user_id": "user-abc123",
    "nickname": "플레이어1",
    "score": 9500,
    "level": 10,
    "created_at": "2026-03-25T00:00:00.000Z"
  }
]
GET /rank/{user_id} 유저 랭킹 조회 
{
  "user_id": "user-abc123",
  "nickname": "플레이어1",
  "score": 9500,
  "level": 10,
  "ranking": 3
}

ℹranking은 매일 자정 갱신되는 스냅샷 기준. 당일 점수는 다음날 반영. 미반영 시 ranking: null

200 OK 404 존재하지 않는 유저
POST /rank 점수 등록

⚠POST /member 로 회원 등록 후 사용 가능

Request Body

필드 타입 필수 설명
user_id string ✅ 등록된 유저 ID
score number ✅ 점수
level number 선택 레벨 
{
  "user_id": "user-abc123",
  "score": 9500,
  "level": 10
}
200 OK 400 필수 필드 누락 500 미등록 유저 / 서버 오류

게임 플로우

POST /member → 회원 등록 (최초 1회)
PLAY 게임 플레이
POST /rank → 점수 업로드
GET /rank → Top 10 확인
GET /rank/{id} → 내 순위 확인
